## 1. 비즈니스 컨텍스트

### 1.1 프로젝트 개요

Visit K (이전 가칭)는 외국인 관광객을 직접 잡으려는 한국 K-beauty 비의료 미용업 매장(미용업 5종 + 자유업 4종)을 위한 다국어 통합 운영 SaaS이다. 8개 모듈(통합 인박스 / 결제 / 예약 / 통역 / 환급 / 후기 / 대시보드 / 마케팅)을 5개 언어(ko / en / ja / zh-CN / zh-TW / vi)로 제공한다.

### 1.2 시장 모멘텀 (네이밍 시점, 2026-04 검증)

- 외국인 환자 2025년 **201만 명** (전년 +72%, 보건복지부 2026.04.24)
- 의료관광 총액 **12.5조 원** (산업연구원 2026.04)
- 부산 1Q 외국인 거래액 **+530%** (크리에이트립 2026.04)
- 비의료 미용업 매장 SaaS = **선두 부재 — 18~24개월 골든 윈도우**

### 1.3 브랜드명 8가지 요건

브랜드명은 다음 8가지 요건을 동시에 충족해야 했다:

1. **다국어 발음 가능성** — 한국어 + 영어 + 일본어 + 중국어(간체/번체) + 베트남어 5개 언어권 자연 발음
2. **한국 매장 사장(40대) 발음 친화** — B2B 고객층 특성
3. **외국인 관광객(B2C 최종 사용자) 발음 친화** — 5개 국적 + 추가 인근국
4. **B2B SaaS 톤** — Stripe / Linear 같은 인프라 미학
5. **환대(Hospitality) 의미 함의** — Intercom / Drift 같은 커뮤니케이션 미학
6. **K-beauty / Salon / SaaS 3중 saturation 회피** — 카테고리 trademark 충돌 방지
7. **글로벌 확장 여지** — Phase 2에서 일본 / 대만 시장 진출 가능성
8. **Jayden 포트폴리오 응집성** — AutoVox · Chatsio · PayLoom · Findably · Teora 와의 시각적/음성학적 동질감 (4-7자 코인 단어)

### 1.4 명시적 회피 영역

- ❌ 직접 "K-" prefix 사용 (글로벌 확장 시 제약 + K-beauty 등록 상표 다수)
- ❌ 살롱/뷰티 카테고리에서 흔한 단어 (Bloom, Glow, Blossom 등)
- ❌ 라틴 어미 -era / -ora / -ura (유럽 InsurTech/SaaS 점유 다수)

## 6. Hesya 의미 & 브랜드 스토리

### 6.1 어원 분해

```
Hesya = Hello + -sya
       ↓
       "안녕"이라는 인사를
       한국어 음운으로 감싸다
```

- **Hello**: 5개 언어권 모두 가장 친숙한 인사 단어 (영어로 보편적이지만 일본어 ハロー, 중국어 哈囉, 베트남어 Halo로도 통용)
- **-sya**: 한국어 종결 음운 (예: "안녕하세요"의 종결, 부드러운 발음 + 한국 전통의 정중함)

### 6.2 브랜드 컨셉

> **"The Korean welcome, in 5 languages"**

외국인이 한국 K-beauty 매장에 들어서는 첫 접점은 **환대(Hospitality)**다. 매장 사장은 영어가 서툴고, 외국인은 한국어가 서툴다. 이 첫 인사의 어색함이 곧 매출 손실로 이어진다 (PRD § 1.1 페인 P-S1, P-S2 참조).

Hesya는 이 첫 인사를 **AI가 다국어로 매끄럽게 감싸주는 인프라**다. 매장은 한국어로 인사하고, 외국인은 자국어로 응답을 받는다. 이 "환대의 매끄러움"이 브랜드의 본질이며, 이름 자체가 그 본질을 담고 있다.

### 6.3 브랜드 태그라인 후보

1. **"Hesya — The Korean welcome, in 5 languages"** (글로벌)
2. **"한국이 외국인에게 건네는 첫 인사, Hesya"** (국내)

## 9. 검증 한계 & 변리사 자문 권장

### 9.1 본 검증의 한계 (Honesty Disclosure)

본 네이밍 검증은 다음 한계 내에서 수행되었다:

1. **KIPRIS·USPTO·EUIPO·WIPO 직접 호출 불가**: Claude는 한국·미국·EU·세계지적재산권기구 상표 DB에 직접 접근 권한이 없다. 검증은 Google 인덱싱 + Crunchbase + LinkedIn + 공식 사이트 직접 확인을 통한 1차 스크리닝이다.
2. **WHOIS 직접 조회 불가**: 도메인 가용성은 Google 검색 결과 + 활성 사이트 부재로 추정한 값이다.
3. **변리사 전문 판단 영역**: 칭호유사 / 관념유사 / 외관유사 등 trademark law 정밀 판단은 변리사의 영역이다.

### 9.2 변리사 자문 필수 체크리스트

KIPRIS·USPTO 출원 전 다음을 변리사를 통해 확정해야 한다:

1. ☐ KIPRIS 정밀 클리어런스 — Class 35(광고/사업) + 42(SaaS) + 44(미용업) 동시 조회
2. ☐ USPTO 정밀 클리어런스 — Class 9(다운로드 소프트웨어) + 35(advertising/business) + 42(SaaS)
3. ☐ EUIPO 클리어런스 (Madrid Protocol 활용 검토)
4. ☐ "Hesya"의 한국어 유사 표기(헤시아 / 헤시야 / 헤샤 / 해시아) 4종 동시 조회
5. ☐ 디자인 상표 등록 (logo + wordmark) 검토

### 9.3 위험 시나리오 백업 계획

만약 변리사 정밀 검증에서 Hesya가 거절될 경우:
- **백업 후보 도출 패턴**: 한국어 어원 + Hello/환대 의미 + 5자 + 자음 클러스터 0
- **즉시 대안**: Round 3로 3~5개 후보 추가 도출 가능
- **거절 사유에 따른 대응**:
  - 칭호유사 → 어미/접두 조정
  - 관념유사 → 의미 재정의
  - 외관유사 → 스펠링 변경

## 11. Compound Knowledge — 향후 네이밍 재사용 자료

본 네이밍 과정에서 축적된 학습은 Jayden의 향후 포트폴리오 확장(예: AutoVox · Chatsio · PayLoom · Findably · Teora · Hesya 다음 7번째 브랜드) 시 재사용 가능한 프레임워크로 정리된다.

### 11.1 핵심 원칙 (Top 5)

1. **3중 saturation 구역 우회 원칙**: 카테고리(예: K-beauty) + 인접 카테고리(예: Salon) + 인프라 카테고리(예: B2B SaaS) 3축으로 trademark 검증
2. **라틴 어미 회피 원칙**: -era / -ora / -ura는 유럽 InsurTech/SaaS가 점유 → 코인 단어 어미로 사용 시 Round 1에서 거절
3. **Korean origin distinctive 원칙**: 한국어 어원의 코인 단어 (예: 緣·終·環·韓) + 영문 wrapping은 글로벌 차별화 + KIPRIS 등록 가능성 모두 높음
4. **K-prefix 회피 원칙**: K-beauty 카테고리에 "K-"로 시작하는 영문 상표는 USPTO에 다수 등록 → 거의 100% 충돌
5. **칭호유사 사전 검증 원칙**: 영문상표 X가 있으면 X로 시작하는 모든 변형(Xa, Xo, Xk 등)이 한국어 발음에서 칭호유사로 거절될 수 있음

### 11.2 10단계 검증 프레임워크 (재사용 가능)

```
1단계: 후보 10개 도출 (4가지 전략별 2~3개)
   ↓
2단계: Google 광범위 검색 (회사 + 도메인 + 카테고리 분석)
   ↓
3단계: Crunchbase / LinkedIn / Tracxn 직접 등재 회사 확인
   ↓
4단계: 공식 사이트 직접 확인 (1차 출처 검증)
   ↓
5단계: KIPRIS / USPTO / EUIPO / WIPO 자체 스크리닝
   ↓
6단계: 다국어 부정 의미 + 발음 안전성
   ↓
7단계: SMILE 5점 평가
   ↓
8단계: TOP 후보 ⚠️ 다운그레이드 위험 재검증
   ↓
9단계: 변리사 정밀 클리어런스 (최종)
   ↓
10단계: 출원 + 도메인 확보 + 소셜 선점 동시
```

### 11.3 절대 금지 사항 (Hard Rules)

- ❌ 변리사 정밀 검증 없이 출원
- ❌ 도메인 .com 미확보 상태에서 브랜드 발표
- ❌ 단일 출처 검증 후 ✅ 판정
- ❌ "유사 회사 있지만 카테고리 다름" 안일하게 판단
- ❌ 한국어 발음 칭호유사 검증 누락
