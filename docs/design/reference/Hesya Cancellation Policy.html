환불은 이렇게 진행돼요

01자동 처리

Hesya가 카드사로 자동 환불 요청을 전송합니다.

02카드사 정산

카드사에서 결제 취소 후 약 3 영업일 내 환불이 표시됩니다.

03알림

환불 완료 시 이메일과 앱 알림으로 안내드립니다.

노쇼 (예약 후 미방문)

예약 후 연락 없이 방문하지 않으시면 결제 금액의 100%가 차감되며, 계정에 1회 노쇼가 기록됩니다. 3회 누적 시 예약이 제한됩니다. 사유가 있으시면 매장 또는 Hesya 고객센터 → 로 연락주세요. 가능한 한 양쪽 모두에게 공정한 해결을 도와드립니다.

자주 묻는 질문

결제 완료 후 24시간 이전엔 매장 메시지로 변경 요청이 가능합니다. 매장이 확정하면 추가 결제 없이 일정만 바뀝니다.
먼저 기존 결제를 환불받은 뒤, 새 결제수단으로 다시 예약하는 방식만 가능합니다. 직접 카드 교체는 지원하지 않습니다.
네, Stripe / Alipay / WeChat Pay 모두 자동 환불을 지원합니다. 결제 시 사용한 동일 수단으로만 환불됩니다.
5 영업일을 초과하시면 고객센터로 연락주시면 카드사 컨택을 도와드립니다. 환불 ID를 함께 알려주시면 더 빨라요.
100% 즉시 환불됩니다. 같은 동네에서 비슷한 시술을 하는 다른 K-Verified 매장 추천도 함께 받으실 수 있습니다.
